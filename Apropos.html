À propos de notre projet Web

Présentation

Notre projet consiste à créer un site web permettant de rechercher des livres par titre et d'obtenir toutes les informations disponibles : auteur, date de publication, résumé, etc. L'objectif est de fournir une interface simple et claire pour tous les utilisateurs.

Comment nous avons réalisé le projet

  • Définition des besoins et des fonctionnalités principales.
  • Création de la structure HTML pour les pages et la navigation.
  • Ajout du style avec CSS pour un design clair, moderne et cohérent avec le reste du site.
  • Écriture de scripts JavaScript pour gérer la recherche et l'affichage des livres.
  • Tests avec des données fictives avant intégration à une API ou une base de données.

Technologies utilisées

  • HTML pour la structure des pages.
  • CSS pour le style et le design.
  • JavaScript pour l’interactivité et la recherche dynamique.
