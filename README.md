# Bookmovie
création d'un site WEB montrant les livres adaptés en séries/films 

## Nouveau style — cosy & moderne

J'ai appliqué une refonte visuelle pour rendre le site plus "cosy" et moderne :

- Palette chaleureuse, dégradés et ombres douces
- Meilleure typographie (Inter + Merriweather via CSS)
- Cartes plus arrondies et interactions subtiles (hover, élévation)
- Mise en page centrée et responsive

Pour voir le rendu localement, ouvrez `Page.html` dans votre navigateur (double-cliquez ou utilisez un serveur local).

Si vous voulez d'autres variantes (thème sombre, animations supplémentaires ou grille au lieu d'un carrousel), dites-moi et je l'implémente rapidement.
