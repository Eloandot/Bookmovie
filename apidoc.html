Documentation de l’API – BoookMovie (Open Library)

BoookMovie utilise l’API Open Library pour récupérer les données sur les livres. Cette API est gratuite et accessible aux développeurs souhaitant intégrer des informations sur les livres dans leurs applications ou sites web.

1. Base URL

Toutes les requêtes utilisent l’URL suivante :

https://openlibrary.org/

2. Endpoints principaux

  • /search.json?q=mot_clé : Recherche de livres par titre, auteur ou sujet.
  • /works/OLID.json : Récupère les détails d’un livre à partir de son OLID (Open Library ID).
  • /authors/OLID.json : Récupère les informations sur un auteur via son OLID.

3. Paramètres courants

  • q : Mot-clé pour la recherche (titre, auteur, ISBN, sujet).
  • limit : Nombre de résultats retournés (par défaut 100).
  • page : Numéro de page pour la pagination.

4. Exemple de requête pour rechercher un livre

GET https://openlibrary.org/search.json?q=harry+potter&limit=5

5. Exemple de récupération des détails d’un livre

GET https://openlibrary.org/works/OL82563W.json

6. Exemple en JavaScript (fetch)

fetch('https://openlibrary.org/search.json?q=harry+potter&limit=5')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error(error));

7. Codes de réponse

  • 200 OK : Requête réussie.
  • 404 Not Found : Ressource introuvable.
  • 500 Internal Server Error : Erreur côté serveur Open Library.

8. Support et documentation officielle

Pour plus de détails sur l’API Open Library, consultez leur documentation officielle.
